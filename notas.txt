*JSONWEBTOKEN:

    - JWT es una libreria para generar un token de autenticacion
    Par ello usamos el metodo jwt.sing()
    PARAMETROS:
    1. payload, son los datos que deseas encryptar pero como objeto {data1,data2...}
    2. una clave secreta
    3. {expiresIn:'<x>h','<x>y','<x>d'}, el tiempo de expirasion
    4. un callback, que filtre un error o el resultado final: (err, token)=>{}

    este proceso lo encapsulamos en una promesa por si algo fallaba

    Para validar un jwt obtenido usamos la funcion verify()

    PARAMETROS:

    1. token obtenido
    2. llave secreta
